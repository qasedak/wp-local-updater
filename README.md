# WP Local Updater

A WordPress plugin for updating WordPress core from a direct download URL or uploaded ZIP file, ideal for environments with restricted internet access.

## Features

- Update WordPress core offline using a manually downloaded ZIP file
- Support for direct URL downloads
- Perfect for restricted server environments
- Simple, intuitive admin interface

## Usage

1. Go to **Tools → Local Updater** in your WordPress admin panel
2. Either upload a WordPress ZIP file or provide a download URL
3. Follow the on-screen instructions to complete the update
